============
Installation
============


Stable release
--------------

To install AWS CFN Resources Schemas, run this command in your terminal:

.. code-block:: console

    $ pip install aws_cfn_resources_schemas

This is the preferred method to install AWS CFN Resources Schemas, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for AWS CFN Resources Schemas can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/compose-x/aws_cfn_resources_schemas

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/compose-x/aws_cfn_resources_schemas/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/compose-x/aws_cfn_resources_schemas
.. _tarball: https://github.com/compose-x/aws_cfn_resources_schemas/tarball/master
